# ⌚ Aura Watches - Luxury E-Commerce UI ( Just Crated For Using In My Portfolio )

یک رابط کاربری فروشگاهی (High-Converting UI) فوق‌العاده لوکس و مدرن که برای فروشگاه تخصصی ساعت‌های اورجینال و هوشمند طراحی شده است. در این پروژه از پالت رنگی سبز زمردی عمیق (Emerald Green) و طلایی مات برای انتقال حس اصالت و قدرت استفاده شده است.

## ✨ ویژگی‌های کلیدی (Features)

- **ساختار High-Converting:** چیدمان مهندسی‌شده برای هدایت سریع کاربر به سمت خرید (دسته‌بندی‌های کپسولی، نوار محصولات پرفروش).
- **طراحی کلاسیک و ماندگار:** استفاده از تایپوگرافی کلاسیک (Cinzel) در کنار فونت‌های مدرن برای ایجاد تضاد بصری جذاب.
- **انیمیشن‌های ورودی (Entrance Animations):** استفاده از GSAP برای لود نرم و سینمایی المان‌ها هنگام اسکرول.
- **کاملاً واکنش‌گرا:** تجربه کاربری بی‌نقص در تمامی دستگاه‌ها (موبایل، تبلت و دسکتاپ).

## 🛠 تکنولوژی‌های استفاده شده (Tech Stack)

- **HTML5** (Semantics)
- **CSS3** (Flexbox, Grid, CSS Variables)
- **Vanilla JavaScript**
- **GSAP & ScrollTrigger**
- **FontAwesome 6** (Icons)

## 🚀 نحوه اجرا (How to Run)

نیازی به نصب پکیج نیست. فایل‌های مخزن را دانلود کرده و `index.html` را در مرورگر خود اجرا کنید. برای عملکرد صحیح انیمیشن‌ها و آیکون‌ها، اتصال به اینترنت الزامی است.
